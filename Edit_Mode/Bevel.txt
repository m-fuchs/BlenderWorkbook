====== Bevel ======

===== Bevel eines einzigen Vertex =====

Auch Vertices können per **Bevel (Ctrl + B)** abgeschrägt werden, man muss im Bevel-Modus nur **V** drücken.
Auch in diesem Modus können per Scrollrad die Level bestimmt werden, mit denen abgeschrägt wird.
