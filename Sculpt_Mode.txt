====== Sculpt Mode ======

===== DynTopo =====
Wenn der **DynTopo**-Modus aktiviert ist (**CTRL + D**) wird beim Zeichnen im **Sculpt-Mode** automatisch mehr Topologie zum
bearbeiteten Objekt hinzugefügt. Die Aktivierung diese Modus ist nicht persistent. Nach jedem neuen Wechsel in den 
**Sculpt-Mode** muss er wieder aktiviert werden.


===== Topologie hinzufügen =====
Im Sculpmodus kann die Topologie leicht verändert werden. Mit **R** wird die Abstandseinstellung der Vertices 
aktiviert. Anschließend kann man mit Scrollrad die Größe der Topologie einstellen. Das eigentliche Re-Topologieren wird
anschließend über **STRG + R** durchgeführt.


===== Objekt auswählen =====
Um ein Objekt im Sculpt Mode aus zuwählen, muss der Cursor über dem Objekt platziert werden und der Shortcut
**Alt + Q** gedrückt werden. Anschließend sollte das neu ausgewählte Objekt kurz orange aufblinken.
